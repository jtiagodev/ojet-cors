# oracle-servicebus-cors

Proxy enabling CORS for Oracle Service Bus

# Install

```
npm i oracle-servicebus-cors
```

# Changelog
v0.1.0 - Initial release

# Documentation

Create a .env file with the following structure:
```
protocol="http"
server="url.com"
port=0000
```

```
npm install
node osb-cors
```
